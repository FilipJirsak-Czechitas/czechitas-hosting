Vercel a Next.js

Vercel je tvůrce Next.js, takže pro hostování Next.js projektů je to nejlepší volba.

GitHub Pages: pages.github.com

  • Vhodné pro jednoduchý web
  • (HTML, CSS, JavaScript – bez WebPacku resp. npm)
  • Obvykle se používá jako doprovodný web pro projekty na GitHubu

Vlastní doména

Pokud nechcete použít *.netlify.app nebo *.vercel.app

  • Zvolit si registrátora, např. SubReg
  • Následně je nutné doménu přesměrovat na hosting – podle návodu hostingu
  • V cenách od stokorun ročně po „prémiové“ domény za desítky tisíc.
  • Pozor na cenu za prodloužení domény po roce!

Na vyzkoušení můžu zdarma poskytnout
